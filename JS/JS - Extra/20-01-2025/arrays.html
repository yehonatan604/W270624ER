1. מציאת ערך מינימלי ומקסימלי

מצא את הערך המינימלי והמקסימלי במערך הבא:

[45, 23, 78, 12, 89, 34, 56, 90, 11, 67]

2. מציאת ערך במערך

מצא את הערך 34 במערך המספרים

3. מציאת אובייקט במערך

מצא את האובייקט של בעל החיים "חתול" במערך הבא:

        { animal: "כלב", age: 5 },
        { animal: "חתול", age: 3 },
        { animal: "ציפור", age: 2 },
        { animal: "ארנב", age: 1 },

4. מיון מערך

מיין את מערך בעלי החיים לפי הגילאים בסדר יורד

5. תרגיל מסכם

לפניכם נתון מערך אשר מכיל אובייקטים של תלמידים וציונים:

          { name: "דניאל", grade: 90 },
          { name: "אביב", grade: 85 },
          { name: "אורי", grade: 70 },
          { name: "אורן", grade: 95 },
          { name: "אורלי", grade: 60 },
  • חלקו את המערך לשני מערכים - נכשלים ועוברים
  • נכשלים - תלמידים שציונם נמוך מ-70 (כולל)
  • עוברים - תלמידים שציונם גבוה מ-70
  • מיינו את המערכים לפי הציונים בסדר עולה
  • הדפיסו את שם התלמידים בעלי הציון הגבוהה ביותר בכל אחד מהמערכים
  • הדפיסו את שם התלמיד בעל הציון הגבוהה ביותר בכיתה
  • היפכו את שם התלמיד כך ששמו יופיע הפוך, למשל: דני - ינד
